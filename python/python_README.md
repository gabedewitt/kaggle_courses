Repositório contendo os exercícios desenvolvidos no curso de Python do Kaggle.
O curso abordou os seguintes tópicos: 
1) Sintaxe, atribuição de variáveis e números
2) Funções e função help
3) Booleanos e condicionais
4) Listas
5) Estruturas de repetição e List Comprehension 
6) Strings e dicionários
7) Uso de bibliotecas externas
